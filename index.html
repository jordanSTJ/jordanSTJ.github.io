some simple code for beginners

basic lines of code to learn is

  • A print statement print("")
  • To create a loop in a piece of code us a (while True:) at the very top of the code
  • To make a variable you type any word then a (=) along with ("") and then whatever you want it to equal
    then when you type the veriable it will pop up whatever you made it equal
  • To added a if, elif and else statement you put in if choice=="whatever": to start it off
    then elif choice=="whatever": statements for every other options
    then to finish it you put in a (else:) statement and a print like print("this is not an option")
  • To create a list you put the name of the list (e.g bag) then a (=)
    with ["name of object then "and a (,) space and another (") then an object and another (") then to finish you put a (])
  • Then to print the items in the bag type (for item in bag:) then press return and type (print (bag))
  • To make the code stop for as long as you want you added a (from time import sleep) above the (while True:)
    then where you want the code to slow down you put a sleep(however long you want)then continue
  • To break a while True loop you just type break under where you what the code to break
